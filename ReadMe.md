
# YorkU Conference Room Scheduler

## Project Overview

The **YorkU Conference Room Scheduler** is a GUI-based Java application designed to support room booking and conference room management for York University.

The full system is intended to support registered users, administrators, and the Chief Event Coordinator. Users can book available rooms, edit or cancel bookings before the start time, extend bookings before expiry if the room is available, and interact with related modules such as payment, profile, room management, and check-in.
